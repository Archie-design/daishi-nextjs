# 大師修顧客評論小助手 — 管理者操作手冊

---

## 系統簡介

本系統提供兩個主要功能：

| 功能 | 說明 |
|------|------|
| **前台**（顧客使用） | 顧客填寫維修體驗 → AI 生成評論草稿 → 一鍵複製前往 Google Maps |
| **後台**（管理者使用） | 管理選項內容、查看評論記錄、下載 QR Code、修改登入密碼 |

---

```
admin888
```

---

## 二、修改登入密碼

1. 登入後台
2. 捲動至頁面最下方「**🔐 修改後台密碼**」區塊
3. 填入「目前密碼」與「新密碼」（至少 4 個字元）
4. 點「**更新密碼**」

> 密碼以雜湊加密後儲存於 Google Sheets 的 Config 工作頁，管理者本人也無法反查原始密碼。

---

## 三、管理前台選項

後台上方四個區塊可自訂前台顯示的選項：

| 區塊 | 說明 |
|------|------|
| ① 今日服務項目 | 顧客選擇的維修類型 |
| ② 手機品牌 | 顧客的手機品牌 |
| ③ 今天最滿意的地方 | 好評關鍵詞 |
| ④ 想要的評論語氣 | AI 生成評論的語氣風格 |

### 新增選項
在輸入框輸入文字 → 按 **Enter** 或點「**+ 新增**」

### 刪除選項
點選項右側的 **×**

### 儲存
修改完成後點「**💾 儲存所有設定**」，前台即時生效。

### 恢復預設值
點「**↺ 恢復預設值**」可還原所有選項至系統預設狀態。

## 四、評論記錄

系統每天早上 9:00 自動從 Google Maps 抓取最新評論，儲存於 Google Sheets 的 Reviews 工作頁。

### 查看方式
後台「**📋 評論記錄**」區塊，顯示三個統計數字：

| 數字 | 說明 |
|------|------|
| 總評論數 | Sheets 中所有評論筆數 |
| 平均評分 | 所有評論的平均星級 |
| 未回覆 | 尚未在 Google Maps 回覆的評論數（紅色警示） |

### 篩選評論
- **全部**：顯示所有評論
- **低分**：只顯示 4 星以下評論
- **未回覆**：只顯示尚未回覆的評論

### 標籤說明
| 標籤 | 說明 |
|------|------|
| 已回覆 | Google Maps 上已有商家回覆 |
| 已示警 | 低分評論已發送 Email 通知 |

> 注意：Places API 備用模式無法自動偵測「已回覆」狀態，此欄位一律顯示為未回覆。

## 五、店內 QR Code

後台「**🔲 店內 QR Code**」區塊提供掃描直達 Google Maps 評論頁的 QR Code。

### 下載與使用
1. 點「**⬇ 下載 PNG**」
2. 印製後放置於店內明顯位置（收銀台、維修等待區）
3. 顧客掃描後直接進入 Google Maps 評論頁

> QR Code 連結固定指向商家 Google Maps 評論頁，不會改變。

## 七、手動觸發監控（進階）

如需立即抓取最新評論（不等每天 9:00），可使用以下指令：

### 立即監控
```bash
curl -X GET "https://daishi-nextjs.vercel.app/api/cron/monitor" \
  -H "Authorization: Bearer <CRON_SECRET>"
```

### 立即產生週報
```bash
curl -X GET "https://daishi-nextjs.vercel.app/api/cron/weekly" \
  -H "Authorization: Bearer <CRON_SECRET>"
```

> `CRON_SECRET` 請向系統管理員索取（儲存於 `.env.local` 或 Vercel 環境變數）。

## 九、常見問題

**Q：評論記錄顯示 0 筆？**
A：系統每天 9:00 自動抓取，或請管理員手動觸發一次監控。

**Q：忘記後台密碼怎麼辦？**
A：至 Google Sheets 的 Config 工作頁，找到 `adminPassword` 那列，將 B 欄的值清空，儲存後系統會在下次登入時恢復為預設密碼 `admin888`。

**Q：Email 通知沒收到？**
A：確認 Gmail 應用程式密碼是否有效，以及信件是否被歸類至垃圾郵件。

**Q：Places API 限制只有 5 則評論？**
A：是的，備用模式每次只能取得 Google Maps 上最新顯示的 5 則評論。若需取得全部評論，需申請 Google Business Profile API 白名單（約 1-3 個工作天）。
